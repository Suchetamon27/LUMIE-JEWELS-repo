## 💎LUMIE JEWELS V.1.0
Intricate Heritage, Modern Luxury
An elegant, high-end Single-Page Application (SPA) showcasing exquisite Indian luxury fine jewelry, designed with a custom aesthetic design.

## 📖 Overview
LUMIE JEWELS is a premium digital storefront built for discovering luxury Indian jewelry. The application prioritizes visual excellence, smooth scrolling transitions, and a clean, spacious layout to emulate the experience of a high-end salon.

Catering to fine jewelry buyers, it displays pricing in Indian Rupees (INR, ₹) and features a comprehensive product discovery catalog. The storefront showcases 18 detailed items—ranging from heavy wedding bridal wear (Rivaah) to sleek everyday lightweight gold & diamonds (Mia)—using authentic retail terminology and verification standards.

## ✨ Features
- Branding: Elegant deep wine burgundy themes combined with warm champagne surfaces and thin antique gold accents.
- Dynamic Search Engine: Live, case-insensitive search bar that filters catalog cards instantly as you type (matching names, types, materials, or collections).
- Multi-Criteria Filter Panel: Combine selections across:
- Sub-Brands/Occasions: Rivaah (Bridal), Mia (Everyday), Astra (Diamonds), and Virasat (Traditional Heritage).
- Product Types: Haar/Necklaces, Jhumka/Earrings, Kada/Bangles, Mangalsutras, Nose Pins, and Gold Coins.
- Metals & Purity: 22KT Gold, 18KT Gold, Diamonds, and Navratna Gemstones.
- Interactive Showcases: Prominent occasion cards linking directly to pre-configured catalog filters with smooth scroll-to-view navigation.
- Luxury Visual Animations:
      Dynamic typewriter effect typing out the main hero header on page load.Slower, cinema-like hover zoom on catalog cards.A diagonal metallic shine sweep that plays across jewelry images upon hover.
- Linear fade-in transitions triggered as elements enter the viewport.
- Concierge Specs Modal: Displays product stories and certified retail specifications (Net Gold weight, gross weight, diamond clarity, and IGI/SGL/BIS hallmark certification details) with a direct dial call-to-action.

## 🛠️ Tech Stack

- Frontend Core:	HTML5	Clean semantic markup structure
- Styling:	CSS3	Vanilla CSS, flexbox/grid layouts, keyframe animations
- Logic:	JavaScript (ES6+)	Vanilla DOM rendering, array filter pipelines, Intersection Observer
- Fonts:	Google Fonts	Montserrat (body text), Playfair Display (luxury headings)
- Icons:	FontAwesome v6	Vector icons for menus, status badges, and actions
- Deployment:	Vercel	Hosted as a zero-configuration static website

## 🚀 Installation & Running Locally
Since this is a lightweight vanilla web project, it runs directly in any modern browser without needing to run compilation or bundler commands.

Step-by-Step Setup
1. Clone the repository
git clone https://github.com/Suchtetamon27/LUMIE-JEWELS-repo.git
2. Navigate into the folder
cd LUMIE-JEWELS
3. Run the local development server:
python serve.py
4. Open in browser: Open http://localhost:8000 in your web browser.
